# 托拉学习:犹太学问的基石

## 引言

托拉(Torah)是犹太生活的核心,是指引犹太民族三千多年的神圣蓝图。托拉不仅仅是一部历史文献,更是一部活的经典——每一代人都在研读、诠释并重新应用它。本指南将向您介绍托拉的结构、内容,以及维系犹太社区的丰富学习传统。

## 什么是托拉?

### 成文托拉(Torah She'bichtav / תּוֹרָה שֶׁבִּכְתָב)

成文托拉,又称**胡马什**(Chumash / חוּמָשׁ)或**摩西五经**,由摩西五书组成:

1. **贝雷希特**(Bereishit / בְּרֵאשִׁית)——创世记
2. **舍莫特**(Shemot / שְׁמוֹת)——出埃及记
3. **瓦伊克拉**(Vayikra / וַיִּקְרָא)——利未记
4. **巴米德巴尔**(Bamidbar / בַּמִּדְבָּר)——民数记
5. **德瓦林**(Devarim / דְּבָרִים)——申命记

根据犹太传统,这些书卷由上帝在西奈山赐予摩西,包含613条诫命(**密茨沃特** / mitzvot / מִצְווֹת),构成犹太律法和实践的基础。会堂使用的托拉经卷(**赛费尔托拉** / Sefer Torah / סֵפֶר תּוֹרָה)由受过训练的文士(**索费尔** / sofer / סוֹפֵר)用特殊墨水手写在羊皮纸上,遵循保持了数千年的精确规则。

### 口传托拉(Torah She'be'al Peh / תּוֹרָה שֶׁבְּעַל פֶּה)

口传托拉包含拉比对成文托拉的大量诠释、解释和应用。根据犹太传统,摩西在西奈山同时接受了成文托拉和口传托拉。口传托拉代代相传,最终被写下来,主要包括:

- **密西拿**(Mishnah / מִשְׁנָה)——约公元200年由拉比耶胡达·哈纳西编纂,包含六部犹太律法
- **革马拉**(Gemara / גְּמָרָא)——拉比对密西拿的讨论和阐释
- **塔木德**(Talmud / תַּלְמוּד)——密西拿和革马拉合在一起(巴比伦和耶路撒冷版本)
- **米德拉什**(Midrash / מִדְרָשׁ)——探索圣经文本更深含义的诠释文学

成文托拉与口传托拉的关系是共生的:成文托拉提供基础,而口传托拉阐明其意义及在日常生活中的应用。

## 五书:主题与关键故事

### 1. 贝雷希特(创世记)——"起初"

**希伯来名称含义:**开篇词"起初"

**中心主题:**
- 世界和人类的创造
- 上帝与人类的关系
- 通过先祖先母追溯犹太民族的起源
- 家庭动态、道德选择和神意

**关键故事:**
- **创世**(第1-7天):宇宙形成,以安息日为顶点
- **亚当和夏娃**:伊甸园与人类的首次道德考验
- **挪亚方舟**:神的审判与通过彩虹之约的更新
- **巴别塔**:人类的分散与民族的诞生
- **亚伯拉罕之旅**:第一位先祖的信仰呼召和割礼之约
- **捆绑以撒(阿克达)**:信仰的终极考验
- **雅各的天梯**:先祖的天使异象和神的应许
- **约瑟和他的兄弟们**:嫉妒、流放与在埃及的救赎

**中国文化相似性:**如同《易经》通过卦象探讨宇宙模式和道德哲学,贝雷希特确立了创世和人类行为的基础模式,这些模式在犹太思想中回响不绝。

### 2. 舍莫特(出埃及记)——"名字"

**希伯来名称含义:**"名字"(源自"这些是……的名字")

**中心主题:**
- 奴役与救赎
- 神的启示与盟约
- 建立圣洁的社群
- 律法与社会正义

**关键故事:**
- **在埃及为奴**:以色列人的压迫与自由的呼声
- **摩西与燃烧的荆棘**:神的呼召和神名的启示
- **十灾**:神力对抗世俗暴政
- **出埃及与红海分开**:从奴役中解放
- **西奈启示**:领受十诫和盟约
- **金牛犊**:考验忠诚与悔改的力量
- **会幕(米什坎)**:为神建造居所

**中国文化相似性:**舍莫特中的解放和社会正义主题与儒家的仁政理想(*仁*)以及孟子的"统治者必须服务人民福祉"概念相呼应。

### 3. 瓦伊克拉(利未记)——"祂呼召"

**希伯来名称含义:**"祂呼召"(上帝呼召摩西)

**中心主题:**
- 圣洁与神圣
- 献祭与赎罪
- 祭司服事
- 仪式洁净与不洁
- 伦理与社会责任

**关键章节:**
- **献祭律法**:各种祭品及其含义
- **祭司授职**:亚伦及其子孙的分别为圣
- **饮食律法(卡舒鲁特)**:许可和禁止的食物
- **赎罪日仪式**:赎罪日的礼仪
- **圣洁法典**:"你们要圣洁,因为我耶和华你们的神是圣洁的"
- **禧年**:每50年一次的经济正义与债务豁免

**中国文化相似性:**对仪式规范和社会和谐的强调呼应儒家的*礼*——维护宇宙和社会秩序的恰当礼仪遵守。

### 4. 巴米德巴尔(民数记)——"在旷野"

**希伯来名称含义:**"在旷野"

**中心主题:**
- 旅程与漂泊
- 考验与信仰
- 社群组织
- 世代更替

**关键故事:**
- **人口普查与部落组织**:为旅程构建社群结构
- **派遣探子进迦南**:信仰失败导致40年的漂泊
- **可拉叛变**:挑战领导及其后果
- **摩西击打磐石**:领导的重担与错误的代价
- **铜蛇**:通过信仰得医治
- **巴兰的祝福**:列国对以色列命运的视角

**中国文化相似性:**旷野之旅反映道家通过艰难转化的主题,类似于*无为*的概念——通过顺服和信靠找到道路。

### 5. 德瓦林(申命记)——"话语"

**希伯来名称含义:**"话语"(源自"这些是……话语")

**中心主题:**
- 摩西最后的教导和告别
- 回顾律法和盟约
- 对上帝的爱与敬畏
- 个人和集体责任
- 进入应许之地

**关键章节:**
- **示玛**:犹太信仰的核心宣告("以色列啊,你要听……")
- **回顾十诫**:以新诠释重述
- **君王、审判官和先知的律法**:治理与领导
- **祝福与诅咒**:盟约忠诚的后果
- **摩西之歌(哈阿齐努)**:神公义的诗意见证
- **摩西之死**:领导权传给约书亚

**中国文化相似性:**摩西作为教师在去世前回顾核心智慧,类似于孔子的《论语》——旨在引导后代正确行为和智慧的教诲汇编。

## 各书的主要密茨沃特

613条诫命(**塔雅格密茨沃特** / Taryag Mitzvot / תרי"ג מצוות)分布在五书中:

### 贝雷希特(创世记)
创世记包含相对较少的明确诫命,因其侧重叙事。然而,它建立了基础原则:
- 生养众多
- 割礼(布里特米拉)
- 不吃坐骨神经(吉德哈纳舍)

**数量:**约3条密茨沃特

### 舍莫特(出埃及记)
出埃及记引入核心诫命:
- 十诫(阿塞雷特哈迪布罗特)
- 守安息日
- 节期遵守(逾越节、七七节、住棚节)
- 希伯来奴仆的律法
- 禁止欺压寄居者
- 建造会幕

**数量:**约111条密茨沃特

### 瓦伊克拉(利未记)
利未记包含最详细的仪式和伦理律法:
- 献祭体系
- 饮食律法(卡舒鲁特)
- 性禁忌
- "爱人如己"(利未记19:18)
- 圣洁法典
- 安息年律法

**数量:**约247条密茨沃特

### 巴米德巴尔(民数记)
民数记包括与旅程和社群相关的律法:
- 祭司祝福
- 拿细耳人的誓愿
- 红母牛洁净
- 繸子(齐齐特)
- 逃城

**数量:**约52条密茨沃特

### 德瓦林(申命记)
申命记回顾并扩展早期律法:
- 示玛宣告
- 经文匣(特菲林)和门框经文(梅祖扎)
- 慈善律法
- 司法体系
- 君王的限制
- 假先知

**数量:**约200条密茨沃特

## 如何学习托拉

托拉学习(**塔木德托拉** / Talmud Torah / תַּלְמוּד תּוֹרָה)被认为是犹太教中最高的密茨沃特之一。传统方法提供多种理解切入点:

### 四个层次:帕德斯(PaRDeS / פַּרְדֵּס)

希伯来缩写词**帕德斯**(意为"果园"或"乐园")代表四个诠释层次:

1. **佩沙特**(Peshat / פְּשָׁט)——*简明/字面意义*
   - 文本的直接阅读
   - 理解叙事和基本诫命
   - 所有其他诠释的基础
   - **例子:**"起初,神创造天地"意味着神开启创造

2. **雷梅兹**(Remez / רֶמֶז)——*暗示/寓意*
   - 发现隐藏的意义和联系
   - 数值价值(格玛特里亚)
   - 对其他文本的暗指
   - **例子:**词语数值揭示更深的联系

3. **德拉什**(Derash / דְּרַשׁ)——*诠释/讲道*
   - 米德拉什诠释
   - 道德和伦理教训
   - 阐明文本的故事
   - **例子:**理解为何托拉以字母*贝特*(ב)而非*阿列夫*(א)开头

4. **索德**(Sod / סוֹד)——*秘密/神秘*
   - 卡巴拉诠释
   - 隐藏的神圣智慧
   - 深层灵性意义
   - **例子:**神名的神秘意义

**中国文化相似性:**这些层次类似传统中国文本诠释,儒家经典被阅读以获得字面意义(*文*)、道德教训(*义*)和更深哲学原理(*理*)。

### 学习方法

#### 哈夫鲁塔(Chavruta / חַבְרוּתָא)——学习伙伴
传统方法涉及配对学习:
- 两个学生一起学习,阅读、提问和辩论
- 培养批判性思维和深度投入
- "铁磨铁,磨出刃来;朋友相感也是如此"(箴言27:17)
- 结合个人准备与协作发现

#### 施乌尔(Shiur / שִׁעוּר)——讲座/课程
- 教师主导的教学
- 系统覆盖材料
- 向经验丰富的学者学习的机会
- 经常与哈夫鲁塔学习结合

#### 独立学习
- 阅读注释(拉希、伊本·以斯拉、拉姆班等)
- 沉思和个人反思
- 记录见解和问题

### 初学者必备注释

1. **拉希**(索罗门·伊茨哈基,1040-1105)——结合佩沙特和米德拉什的基础性清晰解释
2. **伊本·以斯拉**(亚伯拉罕·伊本·以斯拉,1089-1167)——语法和理性方法
3. **拉姆班**(纳赫曼尼德,1194-1270)——哲学和神秘洞见
4. **斯福诺**(俄巴底亚·斯福诺,1475-1550)——伦理和哲学诠释

## 日常学习计划

### 传统周期

#### 每周帕拉沙
- 一周中学习本周的托拉段落
- 传统习俗:用希伯来语读两次,用阿拉姆语译文读一次(什奈因米克拉维埃哈德塔古姆)
- 为安息日托拉诵读做准备

#### 达夫约米(Daf Yomi / דף יומי)——每日塔木德一页
- 每天学习塔木德一页
- 7.5年完成整部巴比伦塔木德
- 全球社群学习同一页

#### 密西拿约米特——每日密西拿
- 每天学习密西拿两章
- 约一年完成全部六部

#### 纳赫约米——每日先知书和著作
- 每天学习先知书(Nevi'im)或著作(Ketuvim)一章
- 约一年完成

### 初学者友好计划

#### 选项1:每周帕拉沙重点(每天20-30分钟)
- **周日:**阅读译文帕拉沙
- **周一:**学习关键经文的拉希注释
- **周二:**阅读现代注释或文章
- **周三:**回顾并反思主要主题
- **周四:**为安息日讨论准备问题
- **周五:**安息日前回顾帕拉沙
- **安息日:**聆听托拉诵读,与家人/社区讨论

#### 选项2:主题学习(每天15-20分钟)
- 选择一个主题(如正义、慈悲、安息日)
- 学习不同书卷中的相关经文
- 阅读注释和现代应用
- 记录个人反思

#### 选项3:混合方法(每天30-45分钟)
- 15分钟:每周帕拉沙
- 15分钟:先贤之言(Pirkei Avot)或其他篇章
- 15分钟:诗篇或犹太思想文本

## 初学者推荐阅读顺序

### 第一阶段:基础(第一年)
1. **从故事开始:**阅读创世记译文及拉希注释
   - 最易接近的书卷
   - 引人入胜的叙事
   - 基础概念

2. **继续出埃及记:**专注于解放叙事和十诫
   - 犹太认同的核心
   - 自由和启示的主题

3. **增加先贤之言:**学习先贤伦理
   - 简短易懂的智慧教导
   - 实用伦理指导

### 第二阶段:扩展(第2-3年)
1. **完成五书:**阅读利未记、民数记和申命记及注释
2. **探索先知书:**选读以赛亚书、耶利米书和十二小先知书
3. **开始密西拿:**从布拉霍特(祝福)篇章开始

### 第三阶段:深化(第3年以上)
1. **塔木德入门:**从易懂的篇章开始(布拉霍特、梅吉拉)
2. **米德拉什拉巴:**探索讲道诠释
3. **犹太哲学:**迈蒙尼德的《迷途指津》、《库扎里》或现代著作

## 托拉学习中的中国文化相似性

### 道(Dao / 道)与托拉的概念
两者都代表"道路"——智慧、伦理生活和宇宙秩序之路。托拉如同道,无法完全用言语捕捉却指引整个生命。

### 儒家正名(正名)
类似于托拉对精确语言和命名力量的强调(神通过话语创造,亚当命名动物)。

### 孝道(孝)
第五诫("孝敬父母")与儒家对家庭尊重和代际责任的强调深深共鸣。

### 学习作为美德
两个传统都提升学习为道德和灵性实践。儒家君子(*君子*)类似托拉学者(*塔尔米德哈哈姆* / talmid chacham / תַּלְמִיד חָכָם)。

### 天地和谐
托拉中人类作为神的伙伴完善创造的概念,呼应中国哲学对天(*天*)、地与人和谐的强调。

### 仪式精确
两个传统都强调谨慎遵守仪式(中国的*礼*;希伯来的*密茨沃特*)对维护宇宙和社会秩序至关重要。

## 结论:永恒的学习

托拉学习不是掌握固定的知识体系,而是参与永恒的对话。每一代都带来新的问题、见解和应用。托拉被比作水——生命所需,所有寻求者都可自由获得,能够适应盛装它的任何容器形状。

塔木德教导:"反复翻转它,因为一切都在其中"(*先贤之言* 5:22)。这份终身学习的邀请反映了犹太教对神圣智慧无穷无尽的确信。无论您每天学习五分钟还是五小时,无论您精通希伯来语还是依赖译文,关键在于持续性和真诚性。

当您开始或深化您的托拉学习之旅时,请记住这些原则:

- **从您所在之处开始:**每个知识层次都有价值
- **定期学习:**持续性比强度更重要
- **提出问题:**疑惑和挑战是学习过程的一部分
- **加入社群:**通过对话和分享增强学习
- **应用教导:**让学习转化您的生活方式
- **保持耐心:**理解在数年数十年中加深
- **保持好奇:**托拉在生命的每个阶段都有秘密可揭示

古代拉比教导说,神通过观看托拉创造了世界。通过学习托拉,我们瞥见存在的神圣蓝图以及我们在持续创造中的角色。这一神圣实践将我们与千年犹太智慧以及今天世界各地参与同一文本的犹太人联系在一起。

愿您的托拉学习甜蜜、启迪并具有转化性。正如我们用希伯来语所说:**חֲזַק חֲזַק וְנִתְחַזֵּק**(*哈扎克哈扎克维尼特哈泽克*)——"要刚强,要刚强,让我们彼此加强!"
