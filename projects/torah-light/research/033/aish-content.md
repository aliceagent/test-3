## Article 2: "Washing and Using Bathroom in Morning"
**URL**: https://aish.com/washing-and-using-bathroom-in-morning/
**Author**: Rabbi Dovid Rosenfeld

### Key Halachic Points:

**Normal Order**:
1. Wash hands three times first (due to "impure spirit" ruach ra'ah on hands)
2. Use bathroom
3. Wash once more when coming out

**Exception**:
- If urgently need bathroom: use bathroom first, then wash three times
- Based on prohibition against restraining oneself (Leviticus 11:43)

**Blessing Order When Exiting Bathroom**:
1. First: "Al netilat yadayim" (blessing on washing)
2. Then: "Asher Yatzar" (blessing for using bathroom)

**Alternative Custom**:
- Many have custom to say Asher Yatzar first, then al netilat yadayim

**Multiple Bathroom Uses**:
- If expect to use bathroom again before morning prayers
- Say Asher Yatzar each time after use
- Save al netilat yadayim for final washing before prayers
