# 用希伯来语阅读托拉:初学者解码指南

**栏目:希伯来语学习**

你已经学会了希伯来字母。你认识一些单词。现在你打开一个托拉卷轴或祈祷书——突然你迷失了。字母模糊在一起,元音标记使你困惑,单词似乎无法解码。别担心。流利地阅读希伯来语需要时间,但通过正确的方法,即使作为初学者,你也可以开始自信地阅读托拉文本。

本指南教你如何**解码**希伯来单词——将它们分解成可识别的部分,使用元音(nikkud)读出它们,理解常见模式,并逐步建立阅读信心。

## 构建块:元音(Nikkud)

希伯来字母是辅音。元音由称为**nikkud**(ניקוד)的小标记表示,它们出现在字母上方、下方或内部。

### 为什么Nikkud重要

在无元音的希伯来语(如报纸或现代书籍)中,读者从上下文推断元音。但托拉、祈祷书和初学者文本包括nikkud——使它们**更容易阅读**。

**没有nikkud的例子**: שלום  
**有nikkud的例子**: שָׁלוֹם (sha-LOME = 和平)

Nikkud准确地告诉你如何发音每个音节。

### 主要元音标记

| 元音名称 | 符号 | 声音 | 例子 |
|---------|------|------|------|
| **Kamatz** | ָ | "ah"(father) | אָב (AHV = 父亲) |
| **Patach** | ַ | "ah"(短) | דַּג (DAHG = 鱼) |
| **Segol** | ֶ | "eh"(bed) | מֶלֶךְ (MEH-lekh = 王) |
| **Tzeirei** | ֵ | "ay"(say) | שֵׁם (SHAYM = 名字) |
| **Chirik** | ִ | "ee"(see) | מִן (MEEN = 从) |
| **Cholam** | ֹ 或 וֹ | "oh"(go) | שָׁלוֹם (sha-LOME = 和平) |

### 如何使用Nikkud

1. **看字母**(辅音声音)
2. **看元音标记**(元音声音)
3. **组合它们**(字母+元音=音节)

**例子**: בָּ = ב (B) + ָ (ah) = **BAH**

### 练习:读出单词

尝试这些简单的单词:

- תּוֹרָה = **ת (T) + וֹ (oh) + ר (R) + ָ (ah)** = toe-RAH (托拉)
- שַׁבָּת = **ש (SH) + ַ (ah) + בּ (B) + ָ (ah) + ת (T)** = sha-BAHT (安息日)
- מִצְוָה = **מ (M) + ִ (ee) + צ (TZ) + ְ (静音) + ו (V) + ָ (ah)** = mitz-VAH (诫命)

看?你可以逐字母解码希伯来单词!

## 希伯来单词结构:词根系统

与英语不同,希伯来单词围绕**三字母词根**(שורשים, *shorashim*)构建。理解词根有助于你识别相关单词。

### 例子:词根 כ-ת-ב (K-T-V) = "写作"

- כָּתַב (ka-TAV) = 他写了
- כּוֹתֵב (ko-TEV) = 他写
- כְּתִיבָה (k'tee-VAH) = 写作(名词)
- מִכְתָּב (mikh-TAV) = 信(书面文件)

所有这些单词共享词根 כ-ת-ב (K-T-V),意思是"写"。

## 逐步解码过程

当你在托拉或祈祷中遇到希伯来单词时,遵循以下步骤:

### 步骤1:识别前缀

看第一个字母。有ו(和)、ה(这)或ב/כ/ל/מ前缀吗?

**例子**: **בְּיוֹם**  
- 前缀: **בְּ** (在)
- 词根词: **יוֹם** (yom = 日)
- 含义: "在这一天"

### 步骤2:隔离词根

删除前缀和后缀以找到3字母词根。

### 步骤3:使用Nikkud读出

逐字母阅读,将辅音与元音标记组合。

### 步骤4:识别模式

动词模式(binyanim)改变含义。常见模式:

- **קָטַל** (ka-TAL):简单过去时("他做了")
- **קוֹטֵל** (ko-TAIL):现在分词("正在做")
- **יִקְטֹל** (yik-TOL):将来时("他将做")

通过练习,你会立即识别这些模式。

## 常见托拉短语练习

让我们解码实际的托拉经文。

### 创世记1:1 – בְּרֵאשִׁית בָּרָא אֱלֹהִים

**בְּרֵאשִׁית** (b'reishit)
- **בְּ** = 在
- **רֵאשִׁית** = 开始
- **翻译**: "起初"

**בָּרָא** (bara)
- 词根: ב-ר-א (B-R-A = 创造)
- 模式: 过去时
- **翻译**: "创造了"

**אֱלֹהִים** (Elohim)
- **翻译**: "上帝"

**完整经文**: "起初,上帝创造了......"

### 申命记6:4 – שְׁמַע יִשְׂרָאֵל

**שְׁמַע** (sh'ma)
- 词根: ש-מ-ע (SH-M-A = 听)
- 模式: 祈使句(命令)
- **翻译**: "听!"或"聆听!"

**יִשְׂרָאֵל** (Yisrael)
- **翻译**: "以色列"

**完整短语**: "听啊,以色列"(Shema祈祷的开始)

你刚刚用希伯来语读了三句最著名的托拉经文!
