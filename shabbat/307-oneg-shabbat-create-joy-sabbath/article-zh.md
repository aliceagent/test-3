# 安息日之乐:如何在安息日创造喜悦

> **题图描述(DALL-E提示词):** 中国传统山水画风格的宁静景象,描绘俯瞰薄雾山峦的安息日餐桌。前景是一张木桌,上面摆放着白色哈拉面包、酒杯和蜡烛,以精致的笔触呈现。中景是松树间的小亭,远处的山峰消失在柔和的灰雾中。包含流水和飞鸟。色调:深靛蓝、柔和灰、温暖的琥珀烛光点缀。氛围:宁静、喜悦、沉思。风格:宋代山水画融合温柔暖意。

## 引言

如果休息不仅被允许,而且被命令,那会怎样?如果愉悦不是放纵,而是宗教义务,那又如何?在大多数传统中,灵修实践涉及自我否定——禁食、苦行、远离世俗的欢乐。犹太教走了一条截然不同的道路。在安息日,喜悦不是可选的,而是诫命。

这个概念叫做*Oneg Shabbat*(עונג שבת)——字面意思是"安息日之乐"或"安息日愉悦"。它出现在以赛亚书58:13中:"你若称安息日为可喜乐的……就以耶和华为乐。"从这节经文,犹太律法推导出一个非凡的原则:享受安息日不仅被允许,而且是必须的。美食、舒适的休息、亲密的连接、美丽的歌声、带来智识愉悦的妥拉学习——所有这些不是对圣洁的干扰,而是圣洁的表达。

这个教导对熟悉儒家思想的人来说特别引人注目,因为自我修养往往强调纪律和克制。但在这里,犹太教说:愉悦本身,当被圣化并指向上帝时,就成为一种崇拜形式。我们如何创造这种喜悦?实践中的Oneg Shabbat是什么样子?让我们一起探索。

## 愉悦的诫命

### 圣经和拉比来源

妥拉命令:"记念安息日,守为圣日"(出埃及记20:8),但是先知们增加了喜悦的维度。以赛亚的话——"称安息日为可喜乐的"——成为几个世纪犹太实践的基础。塔木德(安息日篇118a-b)详细阐述了如何通过身体愉悦来尊荣这一天。

拉比耶胡达·哈纳西,米示拿的编纂者,生活在中国东汉时期(东汉,25-220年),有一个特殊的仪式:每个周五下午,他会沐浴、穿上干净衣服,并说:"来,让我们出去迎接安息日女王。"这个意象很有启发——安息日是皇室,我们准备以应有的荣誉接待她,就像迎接来访的君主。

塔木德讨论了在安息日必须吃的三顿节庆餐,每一顿都是体验Oneg的机会。这些餐食应该包括你能负担得起的最好食物——肉、鱼、酒、新鲜面包。圣贤教导说,即使是穷人也应该尝试在安息日餐桌上添加一些特别的东西,哪怕只是一小道平时不吃的熟食。

### 为什么愉悦在灵性上重要

这引出了一个哲学问题:为什么上帝关心我们是否吃好吃的食物或穿漂亮的衣服?灵性不应该超越物质吗?

犹太教的答案深刻:灵魂和身体不是敌人,而是伙伴。上帝创造了物质世界,并称其为"甚好"(创世记1:31)。否定身体就是拒绝上帝的创造。相反,我们通过将其指向圣洁来*提升*物质。

这类似于道家的阴阳和谐概念(阴阳)——不是为了另一个而否定一个,而是找到平衡和融合。安息日证明了灵性和物质可以统一。当我们在安息日吃美味的食物、与配偶相爱或在安息日下午安然入睡时,我们不是在"休息"宗教。我们在*做*宗教。

## Oneg Shabbat的五个维度

犹太传统确定了几种体验安息日喜悦的关键方式。把这些看作是食谱中的配料——每一种都有助于完整的体验。

### 1. 美味的食物

Oneg Shabbat的核心是三顿节庆餐:周五晚餐、周六午餐,以及在安息日结束前的下午第三餐(*Seudah Shlishit*)。

**是什么让这些餐食特别?**

- **哈拉面包**:两个漂亮的面包,通常编成辫子,象征着周五在沙漠中降下的双倍吗哪(出埃及记16:22-26)。面包应该新鲜美味,而不是事后考虑。

- **酒或葡萄汁**:在酒上诵念Kiddush(圣化祝福),标志着从普通时间到神圣时间的转变。即使孩子也会得到一小杯。

- **你最喜欢的食物**:塔木德说每个人都应该吃自己最喜欢的食物。对一些人来说,这意味着传统的阿什肯纳兹cholent(慢炖炖菜)或塞法迪chamin。对其他人来说,可能意味着中式红烧鱼或烤肉。关键是它给你带来喜悦。

- **甜点**:鼓励吃甜食!蛋糕、水果蜜饯,夏天甚至冰淇淋。安息日不是节食的日子。

这里有一个美丽的中国平行:就像传统的中国年夜饭将家庭聚集在象征富足和好运的菜肴周围一样,安息日餐将家庭团结在充满祝福象征的餐桌周围。

### 2. 休息和睡眠

在我们成就驱动的世界中,休息常常让人觉得像懒惰。但在安息日不是。安息日的*menucha*(休息)本身就是一个诫命。

塔木德将安息日下午的小憩描述为这一天的特殊愉悦之一。在周六午餐后,习俗是休息。这不仅仅是身体恢复——而是灵性的复兴。佐哈尔,犹太神秘主义的奠基文本,写于中世纪西班牙(对应中国宋朝,宋朝,960-1279年),教导说安息日下午是"额外的灵魂"(*neshama yeteira*)进入每个犹太人体内的时候,扩展我们对圣洁和平静的能力。

安息日的睡眠被认为是*oneg*——不是浪费时间,而是神圣的愉悦。许多人发现安息日下午是他们做梦最生动的时候。一些犹太思想家认为这是因为,摆脱了一周的焦虑,灵魂可以更自由地漫游。

### 3. 亲密连接

犹太律法不仅允许而且鼓励在安息日晚上的夫妻亲密关系。事实上,对于经常离家的学者和学生来说,周五晚上被认为是履行这一诫命的理想时间。

为什么这是Oneg Shabbat的一部分?因为丈夫和妻子之间的肉体之爱,当以圣洁表达时,反映了上帝与犹太人之间的神圣关系。雅歌,在逾越节诵读,既被理解为一首情诗,也被理解为上帝对以色列之爱的寓言。

这个教导让许多假设宗教和感性是对立的人感到惊讶。犹太教庆祝两者——但坚持它们必须得到适当的引导。在安息日,当一周的压力被搁置,额外的圣洁充满家庭时,这种连接变得更加有意义。

### 4. 妥拉学习

对许多犹太人来说,安息日最深的喜悦不是来自食物或休息,而是来自学习。由于禁止工作,安息日下午成为社区聚集学习妥拉、分析塔木德段落或探索穆萨尔传统的伦理教导的时间。

对局外人来说,这似乎像是"工作",但对于那些沉浸在犹太学习中的人来说,这是纯粹的快乐。塔木德说,妥拉学习比曾经在圣殿中献祭更为上帝所爱。当你在安息日学习时,你不是在劳作——你在品味。

中国文化中有一个平行:明清时期(明清,1368-1912)的文人发现他们最大的喜悦不是在休闲中,而是在书法、诗歌和哲学讨论等精雅追求中。同样,犹太学者在妥拉的智识和灵性挑战中找到*oneg*。

### 5. 歌声和庆祝

安息日充满歌声。在周五晚上的餐桌上,家庭唱*Shalom Aleichem*(欢迎天使)和*Eishet Chayil*(丈夫赞美妻子的歌,来自箴言31章)。在用餐期间,*zemirot*(安息日歌曲)庆祝这一天的圣洁和喜悦。

在哈西德社区,周六晚上的*melaveh malka*(护送女王)餐以歌唱和讲故事为特色,因为安息日离开了。旋律通常是无词的(*niggunim*)——超越语言的灵魂喜悦的纯粹表达。

安息日的音乐有独特的功能:它提升情绪,团结家庭,并提醒每个人安息日不是普通时间。即使你五音不全,一起唱歌的行为也会改变氛围。

## 在家中创造Oneg Shabbat

如果你是安息日遵守的新手,如何开始培养这种喜悦?这里有实际步骤:

### 1. 提前准备

安息日的平静很大程度上来自准备。塔木德说,那些在周五劳作的人将在安息日吃饭。这意味着:

- **在安息日开始前(周五下午)烹饪你的餐食**。你可以在安息日使用*blech*(放在炉子上的金属板)或热板重新加热食物,但你不能从头开始烹饪。

- **打扫你的家**:整洁、美丽的环境增强*oneg*。鲜花、干净的桌布、擦亮的烛台——这些细节很重要。

- **沐浴并穿得漂亮**:舒尔罕·阿鲁赫(犹太律法法典)规定,应该在周五下午沐浴并穿上特殊的安息日衣服。即使没有人会看到你,也要为女王打扮。

### 2. 调动所有五种感官

- **味觉**:美味的食物和酒
- **嗅觉**:新鲜的哈拉、鲜花,也许是为Havdalah(结束安息日的仪式)保留的香料
- **视觉**:蜡烛、漂亮的盘子、桌子周围家人的面孔
- **听觉**:歌曲、祝福、妥拉讨论、没有机械噪音
- **触觉**:哈拉的质地、酒杯的温暖、祝福期间握手

当所有感官都参与时,安息日的体验变得整体。

### 3. 最小化分散喜悦的事物

Oneg Shabbat不仅仅是增加愉悦——也是消除减少它们的东西。这就是为什么安息日禁止工作(*melachah*)。但这不仅仅是体力劳动;而是任何将工作日压力带入神圣时间的事情。

考虑:

- **没有手机或屏幕**:持续连接使注意力分散。安息日提供25小时不间断的存在。
- **没有购物或差事**:一周的任务可以等。
- **没有紧急问题解决**:除非是生死攸关的紧急情况,否则留到周日。

这创造了哈西德大师所说的*menuchat hanefesh*——灵魂的休息。当心智停止奔跑时,喜悦就有了出现的空间。

### 4. 分享喜悦

Oneg Shabbat在分享时被放大。邀请客人。教你的孩子摆桌子。如果你单身或远离家人,找到一个安息日社区或招待他人。*hachnasat orchim*(款待)的诫命在安息日尤其重要。

有一个塔木德教导(安息日篇119b):"凡以安息日为乐的人,都会得到他们内心的愿望。"圣贤解释说,当你把安息日作为优先事项,将其视为珍贵和喜悦时,上帝会通过实现你最深的愿望来回报。这不是魔法——这是灵性原则,我们尊荣的东西,在我们的生活中成长。

## 品尝来世的一天

塔木德将安息日描述为*me'ein olam haba*——来世的一种味道。这是什么意思?

在犹太末世论中,终极未来不是逃离这个世界,而是其完美。一个战争、贫困和苦难被废除的时代,所有人在和平与对上帝的认识中生活。安息日提供了对那个现实的每周一瞥。有一天,我们停止奋斗,只是*存在*。我们承认世界不仅仅依赖于我们的努力——上帝维持着它。

这是深刻的反文化。现代生活坚持:生产、成就、优化。安息日说:停止。休息。愉悦。你不仅仅是你完成的事情。

## 实际应用

这是一个简单的Oneg Shabbat初学者检查表:

**安息日前(周五下午):**
- [ ] 准备三顿节庆餐
- [ ] 打扫你的家并摆好漂亮的桌子
- [ ] 沐浴并换上安息日衣服
- [ ] 在日落前18分钟点蜡烛(传统上由女性做,但任何人都可以)

**周五晚上:**
- [ ] 唱*Shalom Aleichem*和*Eishet Chayil*
- [ ] 在酒上诵念Kiddush
- [ ] 洗手并说哈拉的祝福
- [ ] 与家人或客人享受悠闲的餐食
- [ ] 分享妥拉的话语或有意义的对话

**安息日:**
- [ ] 参加犹太教堂礼拜(如果可能)
- [ ] 享受节庆午餐
- [ ] 安息日小憩或休息
- [ ] 学习妥拉或阅读犹太书籍
- [ ] 在日落前聚集第三餐(*Seudah Shlishit*)

**周六晚上(安息日结束后):**
- [ ] 进行Havdalah(分离仪式)
- [ ] 反思这个安息日给你带来喜悦的事物
- [ ] 计划如何将这种平静带入即将到来的一周

## 核心要点

- **Oneg Shabbat**——安息日之乐——是妥拉诫命,而非可选附加。
- **愉悦是神圣的**:美食、休息、亲密、学习和歌声在安息日都是崇拜形式。
- **准备带来喜悦**:你在安息日前准备得越多,你在安息日期间体验到的平静就越多。
- **安息日是反文化的**:它教导我们不仅仅是我们的生产力,休息是神圣的。
- **永恒的味道**:安息日提供了对完美世界的每周一瞥。
