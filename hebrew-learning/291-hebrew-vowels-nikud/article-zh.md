## 引言:隐藏的发音之谜

当您第一次看到希伯来文本时,可能会注意到一个令人困惑的现象:同一个词可以以两种截然不同的形式出现。在祈祷书中,您会看到נִקּוּד (nikud)——希伯来字母上方和下方装饰着点和线。但打开托拉卷轴,这些相同的字母却光秃秃的,没有任何装饰。这就像阅读带拼音和不带拼音的中文文本——只不过后果更为严重,因为在希伯来语中,这些微小的符号决定了您说的是"国王"(מֶלֶךְ, melekh)还是"天使"(מַלְאָךְ, mal'akh)。

对于正在学习希伯来语的人,特别是习惯于汉字等表意文字系统的人来说,希伯来语的这种双重性质既迷人又令人沮丧。为什么书面希伯来语有时显示元音,有时又隐藏它们?这些点和线都是什么意思?几千年来,希伯来读者是如何在没有它们的情况下阅读的?

本文将解码nikud系统,追溯其令人惊讶的历史,并向您展示这些不起眼的符号如何成为犹太教最重要的教育创新之一。

## 隐藏的问题:希伯来语最初没有元音

### 古代希伯来文没有书写元音

当托拉首次被写下时——大约三千年前,相当于中国的西周时期(西周,约公元前1046-771年)——希伯来语仅用辅音书写。想象一下用英语阅读时只有字母"cn",您必须弄清楚它是"can"、"cone"、"cane"还是"coin"。这本质上就是古代希伯来语的样子。

例如,"书"这个词被写作ספר——但它是发音为*sefer*、*sifre*、*safar*还是*sippur*?母语者从上下文和口头传统中知道,就像古代中国受过教育的读者可以在没有标点符号的情况下破译古典文本一样。

### 口头传统承载元音

几代人以来,希伯来语的发音都是通过口头方式从老师传给学生的。当孩子学习阅读托拉时,他们不仅学习字母——他们还学习旋律、吟唱和世代相传的精确发音。

这个系统运作得很好……直到犹太人流亡。在第二圣殿被毁(公元70年,相当于中国的东汉时期)之后,犹太社区分散在地中海、美索不达米亚,最终到达地球的各个角落。希伯来语不再是日常口语。几个世纪过去了,犹太社区担心:如果我们失去正确的发音怎么办?如果后代忘记如何阅读我们的神圣文本怎么办?

## 马索拉学者:文本的守护者

### 经文科学家的出现

在公元6世纪至10世纪之间(相当于中国的隋、唐、宋初期),一群被称为**马索拉学者**(Masoretes,源自*masorah*,意为"传统")的犹太学者承担了一项非凡的项目。他们主要在提比里亚(今以色列)和巴比伦尼亚(今伊拉克)工作,这些学者-抄写员开发了一套全面的系统来保存圣经希伯来语的精确发音。

马索拉学者一丝不苟。他们数每一个字母,记录不寻常的拼写,标记经文,并且——最重要的是——发明了一套点和线的系统来表示元音。这不是任意的创造;而是忠实的保存。他们正在编码一个已经有一千年历史的口头传统。

可以把他们想象成语言考古学家,在古老的声音景观消失之前,用书面符号捕捉它。

### 为什么叫"Nikud"?

*Nikud* (נִקּוּד)这个词来自希伯来语词根נ-ק-ד,意思是"用点标记"。单个的符号被称为*n'kudot*(单数:*nekudah*,נְקֻדָּה)。这个完整的系统在学术写作中有时也被称为*niqqud*。

## Nikud系统解码

### 七个主要元音

希伯来语有七个主要元音,每个都由特定的符号表示:

1. **Kamatz ( ָ )** ——发"ah"音,类似"father"  
   例子:שָׁלוֹם (*shalom*,和平)

2. **Patach ( ַ )** ——发短"ah"音,类似"bat"  
   例子:דָּבָר (*davar*,词)

3. **Tzeirei ( ֵ )** ——发"eh"或"ay"音,类似"they"  
   例子:שֵׁם (*shem*,名字)

4. **Segol ( ֶ )** ——发短"eh"音,类似"met"  
   例子:מֶלֶךְ (*melekh*,国王)

5. **Chirik ( ִ )** ——发"ee"音,类似"machine"  
   例子:גִּיל (*gil*,欢乐)

6. **Cholam ( ֹ )** ——发"oh"音,类似"note"  
   例子:שָׁלוֹם (*shalom*,和平——注意ו上的cholam)

7. **Kubutz ( ֻ )** ——发"oo"音,类似"boot"  
   例子:רוּחַ (*ruach*,灵)

还有**Shuruk ( וּ )**,这是附加在字母vav (ו)上的kubutz,也发"oo"音。

### 附加符号:配角

除了基本元音之外,nikud还包括几个其他重要符号:

- **Shva ( ְ )** ——表示沉默或非常短促的"uh"音(像"the"中的'e')。这是最复杂的nikud符号,因为它可以是有声的(*shva na'*)或无声的(*shva nach*)。

- **Chataf元音** ——这些是"简化"元音,是shva与patach、segol或kamatz的组合。它们发出非常短促的声音:chataf-patach ( ֲ )、chataf-segol ( ֱ )、chataf-kamatz ( ֳ )。

- **Dagesh ( ּ )** ——字母内部的点(不在下方或上方),使发音变硬(dagesh kal)或使辅音加倍(dagesh chazak)。例子:בּ = "b"(有dagesh)对比 ב = "v"(无dagesh)。

- **Mappik ( ּ )** ——词尾字母ה (heh)内的点,表示应该发音而不是沉默。

- **Rafe ( ֿ )** ——很少使用的符号,表示没有dagesh。

### 阅读示例:分解一个词

让我们解码单词בְּרֵאשִׁית (*Bereishit*,"起初"):

- **בְּ** = *B'* (bet带shva)
- **רֵ** = *rei* (resh带tzeirei)
- **א** = 沉默(alef承载前一个字母的元音)
- **שִׁ** = *shi* (shin带chirik)
- **ית** = *t* (沉默的yod,然后是tav)

结果:*B'rei-shit* ——托拉的第一个词!

## 为什么有些文本有Nikud而有些没有

### 希伯来文本的层次结构

您会注意到犹太文本中的一个模式:

**总是有nikud:**
- 祈祷书(*siddurim*)
- 希伯来语初学者教材
- 儿童读物
- 诗歌(特别是礼拜诗歌,*piyyutim*)

**有时有nikud:**
- 面向年轻读者的现代希伯来语书籍
- 教育材料
- 带希伯来文本的圣经翻译

**从不有nikud:**
- 托拉卷轴
- 塔木德
- 大多数现代希伯来语报纸和书籍
- 日常以色列文字(短信、标志等)

### 托拉卷轴的例外

这里有一些令人着迷的事实:托拉卷轴——最神圣的犹太文本——*没有*nikud书写,就像古代一样。为什么?

**原因1:神圣传统** ——托拉卷轴必须完全按照古代规范书写。添加nikud将是一种改变。

**原因2:解释的灵活性** ——有时同一个希伯来词可以以多种方式阅读,产生不同的含义。拉比评论家经常探讨这些替代阅读。Nikud会锁定一种发音并消除这种解释的丰富性。

**原因3:口头元素** ——阅读托拉是一种传授的技能。您不只是阅读它;您用传统旋律(*trope*或*ta'amei ha-mikra*)吟唱它。缺少nikud确保了托拉阅读仍然是一个活的传统,代代相传。

这就像中国古典诗歌:书面文字是固定的,但解释、语调和重点在表演中赋予它们生命。

## 现代辩论:以色列儿童应该学习Nikud吗?

在现代以色列,有一个有趣的教育辩论。大多数成人希伯来文本不使用nikud——以色列报纸、小说、路标和政府文件都是用"无点"希伯来语(没有元音的希伯来语)书写的。母语希伯来语使用者只是从上下文中识别单词,就像古代希伯来人一样。

那么孩子们应该花费数年学习nikud,最终却阅读没有它的文本吗?

**支持Nikud阵营认为:**
- 从一开始就建立正确的发音习惯
- 有助于区分看起来相同的单词
- 保存马索拉传统
- 对于祈祷和宗教研究至关重要
- 支持初学者的阅读理解

**反对Nikud阵营认为:**
- 没有它,孩子们可以学得更快
- 母语者不需要元音符号
- 大多数成人阅读材料无论如何都缺少nikud
- 它会产生依赖——最好早点学习上下文阅读

在实践中,大多数以色列小学在前几年用nikud教授阅读,然后随着学生词汇量的扩大逐渐断奶。到三年级或四年级,大多数孩子都过渡到无点文本。

### 对于成人学习者:不同的故事

如果您作为成人学习希伯来语(特别是为了宗教目的),nikud是无价的。您不是具有无意识模式识别的母语者。您需要这些点和线来知道一个词是*melekh*(国王)还是*malkah*(女王),*yeled*(男孩)还是*yalda*(女孩)。

大多数祈祷书、圣经评论和成人教育材料正是出于这个原因包含完整的nikud。

## 学习Nikud的实用技巧

### 从元音开始

不要试图一次记住所有nikud符号。从七个基本元音开始,练习它们直到自动化。使用闪卡,反复书写它们,并将每个与英语音似物联系起来。

### 在上下文中学习

与其孤立地研究nikud,不如在阅读实际单词和短语时学习它。从Shema(שְׁמַע יִשְׂרָאֵל)开始,这是最著名的犹太祈祷。它很短,您会经常使用它,并且包含大多数nikud符号。

### 使用希伯来语儿童读物

5-8岁儿童的希伯来语书籍有完整的nikud和简单的词汇。它们非常适合成人初学者,远不如直接进入祈祷书那么令人生畏。

### 练习大声朗读

Nikud是关于声音的。默读不会训练您的嘴和耳朵。大声朗读——即使您独自一人——并听希伯来祈祷或圣经朗读的录音来检查您的发音。

### 知道何时查阅

即使是高级学生也不会记住每一条nikud规则。如有疑问,请查看带nikud的文本(大多数在线希伯来圣经都有nikud),或使用Sefaria.org等工具,该工具显示带完整元音点的圣经文本。

## 中国联系:跨文化的语音系统

作为普通话使用者,您可能会注意到nikud与中国语音系统发展之间的有趣相似之处。

**注音符号/Bopomofo** ——在20世纪初(1918年,中华民国时期)开发,这个语音系统使用特殊符号来指示普通话发音,类似于nikud指示希伯来语发音的方式。像nikud一样,注音符号与汉字一起出现,供学习者使用,但一旦达到阅读流畅性,通常会被去掉。

**拼音** ——在1950年代引入(中华人民共和国时期),拼音使用罗马字母音译汉字。虽然在结构上与nikud不同(它是单独书写的,而不是作为变音符号),但它服务于类似的目的:保存和教授正确的发音。

两种文化都面临同样的挑战:如何保存非语音文字系统的正确发音?解决方案不同,但潜在的关注是相同的——确保后代能够准确发音他们的神圣或古典文本。

## 为什么今天这很重要

### 超越宗教文本

Nikud不仅用于托拉研究。它用于:

- **现代以色列儿童文学**
- **全球希伯来语教材**
- **以色列诗歌**(其中精确的声音和节奏很重要)
- **消歧**(当两个单词没有元音时看起来相同时)

### 古代与现代之间的桥梁

Nikud系统是与过去的活的联系。当您阅读带nikud的现代以色列儿童读物时,您使用的系统与马索拉学者在一千多年前在提比里亚开发的系统完全相同。帮助中世纪犹太社区保存其发音的相同点现在帮助北京的中国人第一次学习希伯来语。

这是最好的语言保存——不是封存在琥珀中,而是适应为每一代新人服务。

## 要点总结

- **Nikud (נִקּוּד)** 是一套点和线的系统,用于指示希伯来语中的元音和某些发音特征
- **古代希伯来语** 仅用辅音书写;发音通过口头传播了一千多年
- **马索拉学者**(公元6-10世纪)开发了nikud来保存圣经希伯来语的传统发音
- **七个主要元音** 构成nikud系统的核心,每个都有独特的符号
- **神圣文本** 如托拉卷轴从不用nikud书写,保存了古代传统和解释的灵活性
- **现代希伯来语** 通常为母语成人读者省略nikud,但为儿童、学习者和宗教文本包含nikud
- **学习nikud** 对于成人希伯来语学习者和任何研究犹太文本的人都是必不可少的

无论您是在学习阅读祈祷书、准备成人礼,还是只是探索犹太文本,理解nikud都是您解锁希伯来语发音的关键。这些谦逊的点和线承载着您祖先的声音——通过练习,它们将帮助您将自己的声音添加到那个古老的合唱中。
